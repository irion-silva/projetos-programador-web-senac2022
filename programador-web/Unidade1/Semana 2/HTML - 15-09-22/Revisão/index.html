Revisão

A seguir veremos um pouco de HTML. uma linguagem de marcação responsável por estruturar uma página WEB.

  • As tags 'h1' ao 'h6' serve para inserirmos um título na tabela
  • A tag 'p' serve para inserirmos um paragrafo na página web
  • Para inserirmos uma lista na página web devemos saber que há dois tipos, as listas ordenadas e as listas não ordenadas. Para as listas ordenadas usamos a tag 'ol' e para as não ordenadas usamos a tag 'ul'
  • Para inserirmos uma imagem na página é necessário a tag 'img'

A seguir veremos o formato das tags

TAGS PARA QUE SERVE CADA TAG
HTML Serve para criar a estrutura HTML
HEAD Define as configurações da página HTML
META CHARSET Define o padrão UTF para a página
TITLE Serve para definir o título para mostrar na barra superior do navegador
BODY Define o conteúdo da página
IMG Serve para inserir uma imagem na página html
H1 ao H6 Serve para inserir um título
BR Serve para pular uma minha no documento HTML
HR Serve para criar uma linha horizontal
FONT Tag estilizar a fonte
P Serve para inserir um parágrafo
B Estiliza o conteúdo com negrito
I Esiliza o conteúdo com itálico
U Estiliza o conteúdo com um sublinhado
Strong Estiliza o conteúdo com negrito
BIG Estiliza o conteúdo com uma fonte de um tamanho maior
SMALL Estiliza o conteúdo com uma fonte de um tamanho menor
SUP Estiliza o conteúdo com sobrescrito
SUB Estiliza o conteúdo com subscrito
TABLE Serve para estruturar uma tabela
TR Cria uma linha da tabela
TD Delimita os dados de uma linha de tabela
UL Cria uma lista não ordenada
OL Cria uma lista ordenada
LI Cria uma linha da lista
SECTION Delimita uma seção na página
MAIN Onde fica o conteúdo principal. Essa tag normalmente vem abaixo do header
FORM Cria um formulário
INPUT Cria um campo de input(dialógo) para o formulário
LABEL Cria um rótulo (descrição) para um input. Essa tag deve vir antes da tag input.
